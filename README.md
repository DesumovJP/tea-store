### Prerequisites
- Node.js 18+ 
- Yarn package manager

### Installation

1. **Backend (Strapi CMS)**
```bash
cd cms
yarn install
cp .env.example .env
yarn dev
# Access admin panel at http://localhost:1337/admin
```

2. **Frontend (Next.js)**
```bash
cd frontend
yarn install
cp .env.local.example .env.local
yarn dev
# Access website at http://localhost:3000
```
